# AstrBot Plugin - 回声洞

一个专门为aiocqhttp(NapCat)设计的回声洞插件，支持添加，访问，查找，删除等回声洞操作，基于SQLite3

## 功能

### 命令列表
**/ca <字符串>**: 添加一条内容为**<字符串>**的回声洞，并随机选择一句名言一起发送
**/ci <数值>**: 访问编号为**<数字>**的回声洞（具备强大的健壮性，可以有效拦截：非法索引,空索引,字符串索引）
**/cq**: 随机访问一条回声洞（有概率访问到被删除的回声洞）
**/cf <字符串>**: 在数据库中查找所有内容包含**<字符串>**的回声洞
**/mycave 多模式匹配**: 
1. mycave <void> -> 寻找自己发送的前100条回声洞编号
2. mycave <int> < 10000 -> 寻找自己发送的第100 * (<int> - 1) + 1到第100 * <int>条回声洞编号
3. mycave <int> >= 10000-> 寻找账号为<int>的人发送的前100条回声洞编号
4. mycave <int1> <int2> -> 寻找账号为<int1>发送的第100 * (<int2> - 1) + 1到第100 * <int2>条回声洞编号
**/rmcave <数值>**: 删除编号为**<数值>**的回声洞（需要发送者亲自删除或由配置的管理员删除）

### 时间复杂度
(以下n均为**表中记录总数**, log如无特殊注明**底数为2**)
/ca : O(log n)
/ci : O(log n)
/cq : O(log n)
/cf : O(n)

/mycave : O(n)
/rmcave : O(log n)

## 安装

1.  将本插件文件夹克隆或移动到您的 AstrBot 插件目录中
2.  重启 AstrBot 以加载插件
